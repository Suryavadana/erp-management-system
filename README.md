# erp-management-system

## Tech Stack
- Frontend: React (JavaScript)
- Backend: Spring Boot
- Database: MySQL
- Authentication: JWT

## Project Overview
A role-based ERP system for managing employees, projects, and tasks.

## User Roles
- Admin
- Manager
- Employee

## Modules 
- Authentication
- Employee Management
- Project Management
- Task Management
